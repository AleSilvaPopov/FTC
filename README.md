# FTC
1. Teoria de Conjuntos
Implementar funções em linguagem C, com base em conjuntos passados, determine as seguintes
operações.

a. União de Conjuntos;
b. Interseção de Conjuntos;
c. Diferença de Conjuntos.

O programa deve ler dois conjuntos digitados pelo usuário e realizar as operações solicitadas.

2. Relações entre Conjuntos
Implementar uma função em C que, com base em um conjunto passado e a sua relação, deve ser
determinada se a relação é reflexiva, simétrica e transitiva. O programa deve ler o conjunto e,
após, a relação passada.
